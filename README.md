# Портфолио Frontend Разработчика

Современное интерактивное портфолио-визитка, созданное с использованием Next.js, Tailwind CSS и Framer Motion.

## 🚀 Технологии

- **Next.js 14** — React фреймворк для серверного рендеринга и статической генерации
- **Tailwind CSS** — утилитарный CSS фреймворк для быстрой стилизации
- **Framer Motion** — библиотека для создания анимаций
- **React Icons** — коллекция иконок для React
- **Type Animation** — анимация печати текста

## 📋 Функциональность

- ✅ Адаптивный дизайн для всех устройств
- ✅ Плавные анимации и переходы
- ✅ SEO оптимизация (метатеги, Open Graph)
- ✅ Форма обратной связи
- ✅ Фильтрация проектов
- ✅ Секции: Обо мне, Навыки, Проекты, Контакты
- ✅ Интеграция с социальными сетями

## 🛠 Установка и запуск

```bash
# Клонирование репозитория
git clone https://github.com/username/portfolio.git

# Переход в директорию
cd portfolio

# Установка зависимостей
npm install

# Запуск в режиме разработки
npm run dev

# Сборка для продакшена
npm run build

# Запуск продакшен версии
npm run start
